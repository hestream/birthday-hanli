# 🚀 部署指南 - 让李晗黎通过手机访问

## 方案一：GitHub Pages（推荐）⭐

### 步骤1️⃣：创建 GitHub 仓库

1. **登录 GitHub**
   - 访问：https://github.com
   - 如果没有账号，先注册一个（免费）

2. **创建新仓库**
   - 点击右上角 `+` → `New repository`
   - 仓库名称：`birthday-hanli`（随意）
   - 设置为 **Private**（私有，只有你能看到）
   - 不勾选 "Add a README file"
   - 点击 `Create repository`

### 步骤2️⃣：上传代码到 GitHub

打开命令行（在当前 birthday 文件夹），依次执行：

```bash
# 1. 添加远程仓库（替换成你的 GitHub 用户名和仓库名）
git remote add origin https://github.com/你的用户名/birthday-hanli.git

# 2. 重命名分支为 main
git branch -M main

# 3. 推送代码到 GitHub
git push -u origin main
```

**提示**：第一次推送会要求输入 GitHub 用户名和密码

### 步骤3️⃣：启用 GitHub Pages

1. 进入你的 GitHub 仓库页面
2. 点击 `Settings`（设置）
3. 左侧找到 `Pages`
4. 在 "Source" 下拉框选择 `main` 分支
5. 点击 `Save`（保存）
6. 等待 1-2 分钟，会显示网址：
   ```
   https://你的用户名.github.io/birthday-hanli/
   ```

### 步骤4️⃣：访问网站 🎉

- **电脑访问**：直接打开上面的网址
- **手机访问**：
  - 方式1：手机浏览器输入网址
  - 方式2：生成二维码让她扫描
  - 方式3：发送链接给她

## 方案二：Vercel（更快，推荐）✨

### 优势：
- ✅ 部署速度更快（30秒完成）
- ✅ 自动 HTTPS
- ✅ 全球 CDN 加速
- ✅ 完全免费

### 步骤：

1. **访问 Vercel**
   ```
   https://vercel.com
   ```

2. **使用 GitHub 登录**
   - 点击 "Sign Up"（注册）
   - 选择 "Continue with GitHub"
   - 授权 Vercel 访问你的 GitHub

3. **导入项目**
   - 点击 "Add New..." → "Project"
   - 选择刚才创建的 `birthday-hanli` 仓库
   - 点击 "Import"

4. **配置项目**
   - Framework Preset: 选择 "Other"
   - Root Directory: `./`
   - 点击 "Deploy"

5. **获取网址**
   - 部署完成后会显示网址（例如）：
   ```
   https://birthday-hanli.vercel.app
   ```
   - 这个网址可以直接通过手机访问！

---

## 方案三：Netlify（简单）🌐

### 步骤：

1. **访问 Netlify**
   ```
   https://app.netlify.com
   ```

2. **拖拽部署**
   - 注册/登录账号
   - 点击 "Add new site" → "Deploy manually"
   - 直接拖拽整个 `birthday` 文件夹
   - 等待上传完成

3. **获取网址**
   - 会自动生成一个网址，例如：
   ```
   https://random-name-123.netlify.app
   ```
   - 可以在设置中自定义域名

## ⚠️ 重要提示

1. **照片隐私**
   - 如果使用 GitHub Pages 私有仓库，需要登录才能访问
   - 建议使用 Vercel/Netlify 公开部署，但不要分享给其他人

2. **域名自定义**
   - 可以购买自定义域名（例如：hanli-birthday.com）

3. **更新网站**
   - 修改代码后，重新 `git push` 即可自动更新
