# AI音乐节奏游戏 - 微信小游戏版

一个结合AI动态生成节奏的音乐游戏，使用**微信小游戏**开发。

## 🎵 功能特点

- 🤖 **AI动态生成节奏** - 每次游戏都不同
- 🎨 **炫酷视觉效果** - 粒子、光效、动态背景
- 📱 **简单触摸操作** - 点击轨道即可
- 🎵 **实时音效反馈** - 每个音轨不同音高
- 📊 **智能难度调整** - AI根据表现自动调整
- ⚡ **能量系统** - 保持连击维持能量
- 🏆 **详细统计** - Perfect/Good/OK/Miss计数

## 🎮 游戏规则

- 4条音轨，音符从上往下落
- 当音符到达判定区时点击对应轨道
- **Perfect**：击中金色区域（30px范围）+100分
- **Good**：击中绿色区域（60px范围）+50分
- **OK**：击中判定区（90px范围）+20分
- **Miss**：未击中或错过 -5能量
- 连击Combo加成：每10连击增加50%分数
- 能量归零游戏结束

## 🎯 音符类型

- 🔵 **普通音符** - 圆形，基础分数
- 💜 **长音符** - 矩形，需要精准timing
- ⭐ **特殊音符** - 星形，额外+50分

## 🎚️ 难度系统

### 简单 (Easy)
- 起始BPM: 100
- 音符密度: 40%
- 适合新手练习

### 普通 (Normal)
- 起始BPM: 120
- 音符密度: 60%
- 平衡的挑战

### 困难 (Hard)
- 起始BPM: 140
- 音符密度: 80%
- 高速连击考验

## 🤖 AI特性

1. **动态节奏生成**
   - AI生成8小节循环模式
   - 强弱拍智能分配
   - 随机组合避免重复

2. **自适应难度**
   - 表现好：BPM +5（最高180）
   - 表现差：BPM -5（最低80）
   - 每500分评估一次

3. **动态强度**
   - 波浪式强度变化
   - 模拟真实音乐起伏

## 📊 评分系统

- **Perfect**: 100分 × Combo加成
- **Good**: 50分 × Combo加成
- **OK**: 20分 × Combo加成
- **Combo加成**: 1 + floor(Combo/10) × 0.5

## 🎮 操作说明

1. **开始游戏**：点击屏幕
2. **选择难度**：点击难度按钮
3. **游戏中**：点击对应轨道击打音符
4. **技巧**：盯住判定线，提前预判

## 如何运行

1. 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 选择"导入项目"
4. 选择本项目目录
5. AppID 使用"测试号"或你自己的 AppID
6. 点击"确定"开始体验游戏

## 技术栈

- 微信小程序原生开发
- Canvas 2D API
- 触摸事件处理
- 震动反馈 API

## 开发说明

本游戏使用微信小程序原生框架开发，主要使用了：

- `Canvas 2D` 进行游戏画面渲染
- `requestAnimationFrame` 实现流畅的游戏循环
- 触摸事件实现挡板控制
- 震动 API 增强游戏反馈
